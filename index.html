The Savage Editor

The lightweight svg editor for annotation on mobile and desktop.

Why?

I wanted an easy to use and pluggable image annotator, but I didn't find anything that fitted my needs. So this is my attempt at writing one.

This plugin is used in a meteor application, if you want to check out that package, go to: "soon"
